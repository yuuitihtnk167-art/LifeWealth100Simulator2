# LifeWealth100 Simulator

マネーフォワードの資産CSVと手入力の前提条件をもとに、100歳までの世帯資産を予測するローカル完結型Webアプリです。

## できること

- 誕生日、フェーズ開始年齢、前提値の入力
- `資産一覧.csv` と `資産推移月次.csv` の取込
- 収入・支出の手入力
- 債券、投資信託・ETF、株式、保険、ドル積立、年金、負債の管理
- バックアップ / 復元
- 調査ページでの警告一覧、月次推移、実績・予測グラフ表示

## 使い方

1. このフォルダをそのままブラウザで開くか、ローカルHTTPサーバーで配信します。
2. `index.html` を開きます。
3. 誕生日を入力します。
4. マネーフォワードの2つのCSVを読み込みます。
5. 各画面で不足条件を補完します。
6. 調査ページで100歳までの推移を確認します。

## ファイル

- `index.html` : 画面構成
- `styles.css` : UIスタイル
- `app.js` : CSV解析、状態管理、シミュレーション、描画

## 補足

- データはブラウザの `localStorage` に保存されます。
- バックアップはJSON 1ファイルで保存します。
- 依存ライブラリは使っていません。
